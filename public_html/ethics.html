Ethics

A set of principles set out to regulate the use of computers. Plagiarism is one of the more common unethical action. It is when a person takes another person's work and claims it as their own without paying respect to the original.

Types of softwares

Free software

These software are freely accessible. You are able to download it's source code and modify it however one wishes.
For example: Scribus, Abiword etc

Freeware

These are software that a user can download free of charge. Although users are not allowed to modify it, only allowed for usage. Subject to all copyright laws
For example: MS Word, Internet Explorer etc

Shareware

These are softwares that are partially usable free of charge. Often the full versions are blocked behind a paywall. The softwares are allowed to be shared only if permission is obtained. Subject to all copyright laws.
For example: IDM, MS Office etc.
