Communication & internet

Transmission of data

Asynchronous data transmission

It refers to data being trasnmitted in an agreed bit pattern. Data bits are grouped together & sent with control bits. This means the receiver of the data knows when the data starts and ends, prevents data getting mixed up

Synchronous data transmission

It is a continuous stream of data (not in discrete groups like asynchronous). Ensures that the sender and receiver are synchronised with each other. Hence, making it the faster method.

Serial & parallel transmission

Serial data transmission

It's when data is sent one bit at a time over a single wire. It works well over long distances. Though, data is transmitted at a slower rate

Parallel data transmission

It's when data several bits (1 byte) are sent down several wires at the same time. It works well over a short distance and it is the faster method.

Simplex, half-duplex and full-duplex

Simplex data transmission

Signal is transmitted in one direction only.
For example: Computer to printer.

Half-duplex transmission

Signal is transmitted in both direction but not at the same time.
For example: Phone conversations.

Full-duplex data transmission

Signal is transmitted in both directions simultaneously.
For example: Broadband connection.

Universal serial bus (USB)

USB is an asynchronous serial data transmission method.

Security aspects

Hacking

It is the act of gaining illegal access to a computer system.
It can lead to identity theft and loss of personal information. To prevent hacking one can use Firewalls, strong passwords and use of anti-hacking software.

Hacking breaks into a computer system to steal data whereas cracking is where one edits a program code for malicious reasons.

Viruses

Program that can replicate itself with the intention of deleting or corrupting files, cause computer malfunction.
It can lead to a device to crashing and deletion or corruption of data. To prevent this one can install an authentic anti-virus software.

Wardriving

The act of locating and using wireless internet connections illegally.
An attacker can hack into wireless network and steal user's password and steal internet time. To prevent this one can use firewalls and strong password.

Internet principles of operation

Internet protocol (IP) address

Each device on the internet is given a unique address known as the IP address. 32-bit number written in the form: 109.108.158.1 . IP address gives the location of a device on the internet whereas the MAC address identifies the device connected to the internet. IP address changes but MAC address remains unchanged.

HyperText Mark-up Language (HTML)

It is used when writing and developing pages. Mark-up language is used in the processing. Html is used to bracket piece of codes. Different intensity of colours is determined by its hexadecimal value

Media Acces Control (MAC)

MAC address refers to a number which uniquely identifies a device on the internet. Refers to the network interface card (NIC) which is part of the device. Usually made up of 48 bits shown as six groups of hexadecimal digits.
NN:NN:NN:DD:DD:DD
(NN:NN:NN) first half is the identity number of the manufacturer of the device.
(DD:DD:DD) second half is the serial number of the device.


Types of MAC address:

  • Universally Administrated MAC Address (UAA)
  • Locally Administrated MAC Address (LAA)

    • UAA is the most common type set by the manufacturer.

    Web addresses

    Each character on the keyboard has its own ASCII code. It can be represented using hexadecimal or decimal values. Hexadecimal addresses are used in the address of files or webpages as a security figure. Takes longer to type in URL but advantage is that you are unlikely to fall into the trap of copying a fake website

    Cookies

    A packet of information sent by a web server to a web browser. Generated each time the user visits the website. Every time a user visits a website, cookies will have collected some key information about the user. They are able to carry out user tracking and maintain user preferences. Although cookies are pieces of data but does not gather personal information.
